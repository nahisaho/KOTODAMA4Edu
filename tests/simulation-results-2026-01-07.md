## 教育的アプローチ（理論的根拠）

以下の理論・フレームワークを適用します：

### 1. ARCSモデル（Keller）
動機づけの4要素で授業を設計：
- **Attention（注意）**: 授業冒頭で「驚き」「謎」を提示
- **Relevance（関連性）**: 現代との繋がりを示す
- **Confidence（自信）**: 達成可能な課題を設定
- **Satisfaction（満足感）**: 学びの成果を実感させる

### 2. 認知負荷理論
- 50分の説明は認知負荷が高すぎる
- 10-15分ごとに活動を切り替える「チャンク化」

### 3. アクティブラーニング
- Think-Pair-Share で全員参加を促す
- 「問い」から始める探究型授業

## 具体的な学習活動（50分授業案）

### 導入（5分）- Attention
「謎かけ」で始める：
```
「この写真は150年前の日本です。
 なぜ武士がいなくなったのでしょう？
 今日はその謎を解き明かします」
```

### 展開1（10分）- Input
- 簡潔な説明（板書は最小限）
- キーワード3つに絞る

### 展開2（10分）- Think-Pair-Share
```
[問い] 「もしあなたが当時の農民だったら、
       この改革に賛成？反対？」
- 1分: 個人で考える
- 2分: ペアで意見交換
- 3分: 全体共有（指名なし、挙手制）
```

### 展開3（15分）- 探究活動
- グループで資料を読み解く
- 「なぜカード」に疑問を書き出す

### まとめ（10分）- Satisfaction
- 今日の「なぜ？」ベスト3を発表
- 次回への「続きの謎」を提示

## 実施上の留意点

1. **準備の負担軽減**: 
   - Think-Pair-Shareは準備不要で即実践可能
   - 「問い」を1つ用意するだけでOK

2. **段階的導入**:
   - 最初は1回の授業に1つのアクティビティから
   - 慣れてきたら増やす

3. **多様な学習者への配慮**:
   - ペア活動は発言が苦手な生徒も参加しやすい
   - 書く活動と話す活動を組み合わせる
